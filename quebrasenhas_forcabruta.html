Quebra de Senhas com Força Bruta

1. O Que é um Ataque de Força Bruta?

Força bruta é um método de ataque onde um invasor testa todas as possíveis combinações de senhas até encontrar a correta.

2. Como Funciona?

O ataque pode ser feito de duas formas principais:

  • Ataque baseado em dicionário: Testa senhas comuns armazenadas em listas.
  • Ataque exaustivo: Gera todas as combinações possíveis de caracteres.

3. Código em Python: Simulando um Ataque

Este script simula um ataque de força bruta contra uma senha **hash** usando MD5.

import hashlib # Senha alvo (armazenada como hash MD5) senha_alvo = hashlib.md5("segura123".encode()).hexdigest() print(f"Hash da senha alvo: {senha_alvo}") # Lista de senhas comuns dicionario = ["123456", "senha123", "admin", "segura123", "qwerty"] # Testando senhas do dicionário for senha in dicionario: hash_senha = hashlib.md5(senha.encode()).hexdigest() print(f"Tentando: {senha} -> {hash_senha}") if hash_senha == senha_alvo: print(f"Senha encontrada! A senha é: {senha}") break else: print("Senha não encontrada no dicionário.")

4. Estratégias de Mitigação

Para proteger sistemas contra ataques de força bruta:

  • Usar hashing seguro: Evite MD5 ou SHA-1, prefira bcrypt ou Argon2.
  • Implementar limite de tentativas: Após várias tentativas inválidas, bloquear o usuário.
  • Usar autenticação multifator: Senhas fortes combinadas com métodos secundários.
