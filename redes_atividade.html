Cenário de Rede - Cisco Packet Tracer

Topo da Rede

Uma empresa fictícia chamada Tech Solutions com três departamentos:

  • TI: VLAN 10
  • Vendas: VLAN 20
  • RH: VLAN 30

Configuração de VLAN

Conecte os switches a um roteador e implemente o roteamento entre VLANs.

Firewall

Adicione um firewall para proteger o tráfego externo e interno, com regras específicas:

  • Permitir HTTP/HTTPS para o servidor web interno.
  • Bloquear tráfego de portas não utilizadas.

Servidor

Configure um servidor com serviços múltiplos:

  • Web Server: Site interno da empresa.
  • DHCP Server: Distribuição de IPs.
  • File Server: Compartilhamento de arquivos.

Dispositivos Finais

Adicione PCs conectados a cada VLAN, com IPs distribuídos pelo DHCP.

Testes

Realize os seguintes testes:

  • Acesso ao site interno via navegador.
  • Verificação de restrições de firewall.
  • Testar conectividade entre VLANs com ping.
