Relatório: IP, Redes e Sub-redes

1. O que é IP?

O Internet Protocol (IP) é um conjunto de regras utilizado para endereçar e encaminhar pacotes de dados entre dispositivos conectados a uma rede. Ele é essencial para permitir a comunicação em redes locais e na internet.

2. Funcionamento do IP

  • Endereçamento: Cada dispositivo recebe um endereço IP único para identificar sua posição na rede.
  • Encapsulamento: Os dados são divididos em pacotes contendo informações como endereço de origem e destino.
  • Roteamento: Os pacotes passam por roteadores, que decidem o caminho mais eficiente para seu destino.

3. Estrutura de Endereços IP

IPv4: Composto por 32 bits, dividido em quatro octetos. Exemplos: 192.168.0.1.

IPv6: Composto por 128 bits, com maior capacidade de endereçamento. Exemplo: 2001:0db8:85a3:0000:0000:8a2e:0370:7334.

4. Tipos de Endereços IP

  • IP Privado: Usado em redes internas. Exemplo: 192.168.1.10.
  • IP Público: Usado para comunicação global na internet. Exemplo: 203.0.113.5.
  • Endereço de Loopback: Para testes no próprio dispositivo. Exemplo: 127.0.0.1.
  • IP Dinâmico: Atribuído automaticamente. Exemplo: 192.168.1.25.
  • IP Estático: Configurado manualmente e fixo. Exemplo: 192.168.1.100.

5. Máscaras de Sub-rede e CIDR

Máscara de Sub-rede: Define qual parte do endereço IP pertence à rede e qual parte identifica os dispositivos. Exemplo: 255.255.255.0.

CIDR: Permite dividir redes em sub-redes menores para maior eficiência. Exemplo: 192.168.1.0/26.

6. Redes e Sub-redes

Uma rede é um conjunto de dispositivos interconectados que compartilham informações. Sub-redes são divisões menores de uma rede maior, ajudando a organizar e reduzir o tráfego.

7. Exemplo Prático: Divisão de Sub-redes

Imagine uma rede original 192.168.1.0/24, dividida em 4 sub-redes menores:

Sub-rede 1 192.168.1.0 192.168.1.1 a 192.168.1.62 192.168.1.63
Sub-rede 2 192.168.1.64 192.168.1.65 a 192.168.1.126 192.168.1.127
Sub-rede 3 192.168.1.128 192.168.1.129 a 192.168.1.190 192.168.1.191
Sub-rede 4 192.168.1.192 192.168.1.193 a 192.168.1.254 192.168.1.255

8. Exemplo Completo: Comunicação em Redes

Um dispositivo com IP 192.168.1.10 envia uma mensagem para outro dispositivo na sub-rede 192.168.1.64, como 192.168.1.70. O roteador utiliza a máscara de sub-rede para identificar origem e destino, garantindo a entrega correta.

9. Conclusão

O IP, juntamente com redes e sub-redes, é fundamental para a organização e eficiência da comunicação entre dispositivos conectados. Compreender esses conceitos é essencial para criar sistemas otimizados e seguros.
